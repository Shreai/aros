Install the AROS Connector

Run the connector on a Mac, Windows PC, or Linux computer that stays on the same local network as Verifone Commander. You will need the one-time activation code shown in AROS.

Before installation

In AROS, add your store, choose Verifone Commander, and generate an activation code.
Use a store computer that can reach Commander. The Commander address may differ by store; the connector discovers and validates the supported configuration page locally.
Keep the AROS onboarding page open while it verifies enrollment, cloud connection, Commander connection, and the first read-only sync.

Security model

  • The activation code is short-lived and single-use.
  • The device credential is stored in the operating-system credential store.
  • Commander access starts in read-only mode.
  • Commander credentials stay on the store computer. Do not enter them on a cloud page.
  • Optional remote administration requires explicit approval and a Cloudflare Access allowlist; Commander is never exposed directly.

Pilot requirement: Node.js 20 or newer, same-LAN access to Commander, and outbound HTTPS to AROS. A bundled runtime will replace the Node.js prerequisite in the signed release.
